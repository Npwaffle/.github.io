GUI HW 3: Multiplication Table

Created by: Nicholas Park

Details:

Please enter a minimum value and maximum value in the column and row sections and then press "Generate Table" to create a table displaying the values and their results when multiplied together.

  • If a value is not entered then an unentered minimum value will default to -12 and an unentered maximum value will default to 12.
  • If an invalid value is entered then the default value will be used instead.
  • If a value with a decimal is entered the value will be rounded down to a whole number.
  • If the minimum value is larger than the maximum value then the values will be swapped
